# Environments management

## Create environment
#### (Python3 version)
`python3 -m venv .py3`

## Create environment
#### (Python2 version)
`sudo pip install virtualenv`
`virtualenv .py2`

## Activate environment
`source .py3/bin/activate`

## Deactivate environment
`deactivate`

# Prepare App
`pip install -r requirements.txt`
